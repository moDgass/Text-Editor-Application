# Text Editor Application
This express application implements a browser text editor with a variety of styling features

# Running Inside Powershell
1. Install dependencies with 
npm install
2. Start the server with 
node app.js
3. Open http://localhost:3000/ in your browser
🚀 Features
🎨 Styling & Formatting
- Font Family Selection — Choose between three distinct font types
- Font Size Controls — Increase/decrease buttons + dropdown selector
- Text Formatting — Bold, italic, underline
- Text Alignment — Left, center, right, justify
🎛️ Color & Theme Controls
- Text Color Picker — Custom RGB palette
- Background Color Picker — Uses the same modular palette logic
- Theme Selector — Applies themes to the editor panel (not the text)
🔍 Find & Replace
- Hidden search panel revealed via button
- Supports keyword search and replacement
🖼️ Image Insertion
- Insert an online image by URL
- Displayed neatly below the editor
📊 Live Counters
- Character count
- Word count
- Line count
- Editing time (minutes + seconds)
📐 Layout Controls
- Margin adjustment
- Padding adjustment
